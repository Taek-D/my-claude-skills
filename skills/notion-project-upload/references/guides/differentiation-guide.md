# Differentiation & Impact Quantification Guide

## 차별화 전략

### 핵심 원칙
포트폴리오 내 프로젝트 3-4개가 동일한 구조로 보이면 "템플릿 사용" 느낌. 각 프로젝트마다 아래 요소를 다르게:

### 1. 강조점 다르게

**성과 중심** (매출, 전환율):
```
## 📊 Data & Metrics (맨 위에!)
• **+23% revenue** - A/B 테스트로 최적 가격 발견
```

**효율 중심** (시간, 비용 절감):
```
## 📊 Data & Metrics
• **-40% processing time** - 자동화 파이프라인 구축
```

**인사이트 중심** (패턴, 세그먼트):
```
## 📊 Data & Metrics
• **3개 유저 세그먼트** 발견 - 각각 다른 전략 필요
```

### 2. 섹션 순서 변형

| 전략 | 순서 | 적합한 프로젝트 |
|------|------|---------------|
| 기본 (추천) | Goal → Dataset → Takeaways → Collaboration → Details | 대부분 |
| 성과 최우선 | Deployment → Goal → Details → Takeaways | 운영 성과 큰 프로젝트 |
| 학습 최우선 | Takeaways → Goal → Details → Collaboration | 주니어, 커리어 전환 |
| 협업 최우선 | Goal → Collaboration → Details → Takeaways | 팀워크 강조 |

⚠️ 순서 변형은 특별한 이유가 있을 때만. 기본 순서가 F-Pattern 최적화됨.

### 3. 문체 다양화

- **Formal**: "본 프로젝트는 이탈률 감소를 목표로 코호트 분석을 수행하였다."
- **Casual**: "왜 유저들이 2주차에 많이 이탈할까? 궁금해서 시작한 분석."
- **Technical**: "FastAPI + PostgreSQL + Redis 스택으로 API 구축."

### 4. 차별화 체크리스트

- [ ] 다른 프로젝트와 다른 지표를 강조하는가?
- [ ] 다른 강조점 (성과/학습/협업)을 선택했는가?
- [ ] 섹션 순서에 변형이 있는가?
- [ ] 시각적 요소 (표/차트 스타일)가 다른가?
- [ ] Extra 섹션으로 고유 정보를 추가했는가?

## 임팩트 정량화 공식

### 1. 시간 절약 → 비용 절감
```
절감 비용 = 절약 시간(h/주) × 인원 × 52주 × 시간당 인건비
```
예: 주 2.5시간 절약 × 5명 × 52주 × ₩30,000 = **연 ₩19,500,000**

### 2. 프로세스 효율 향상
```
효율 향상 = (개선 후 - 개선 전) / 개선 전 × 100%
```
예: 처리 시간 8시간 → 45분 = **-91% 개선**

### 3. 전환율/매출 개선
```
매출 증대 = 전환율 차이 × 월간 트래픽 × 평균 객단가
```
예: +0.8%p × 5,000명 × ₩80,000 = **월 ₩3,200,000**

### 4. 비용 절감
```
비용 절감 = 개선 전 비용 - 개선 후 비용
절감률 = 절감액 / 개선 전 비용 × 100%
```
예: 월 ₩500,000 → ₩100,000 = **-80% 절감**

### 5. 오류/이탈 감소
```
개선 효과 = (개선 전 오류율 - 개선 후 오류율) / 개선 전 오류율 × 100%
```

## 정량화 팁

### 추정이 어려울 때
- **보수적 추정**: "최소한 주 2시간은 절약" (실제 5시간이어도)
- **범위 제시**: "월 ₩200만~₩500만 절감 추정"
- **단위당 가치**: "건당 처리 시간 8분 → 2분"

### 데이터 없을 때
- **업계 벤치마크**: "데이터 분석가 평균 시급 ₩30,000 기준"
- **가정 명시**: "월 방문자 5,000명 가정 시"
- **상대적 개선만**: "처리 시간 -60% 개선"

## 임팩트 서술

**✅ Good**:
- "API 응답 속도 **-84%** (8.2초 → 1.3초)"
- "데이터 정제 자동화로 팀 주 12시간 절약, 연간 **₩18M** 비용 절감"

**❌ Bad**:
- "성능이 많이 좋아졌어요" (정량화 없음)
- "비용을 절감했습니다" (얼마나?)
- "사용자들이 만족했습니다" (측정 지표 없음)
